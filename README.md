<section id="jenis">
  <div class="card">
    <h2>Jenis-Jenis Kejahatan Cyber</h2>
    <ul>
      <li><b>Penipuan Online:</b> Penipuan online adalah tindakan menipu seseorang melalui internet, misalnya jual beli palsu atau meminta transfer uang.</li>

      <li><b>Phishing:</b> Phishing adalah upaya mencuri data pribadi dengan menggunakan link palsu yang menyerupai website asli.</li>

      <li><b>Hacking:</b> Hacking adalah tindakan masuk ke akun atau sistem orang lain tanpa izin.</li>

      <li><b>Malware:</b> Malware adalah program berbahaya seperti virus yang dapat merusak perangkat atau mencuri data.</li>

      <li><b>Cyberbullying:</b> Cyberbullying adalah perundungan yang dilakukan melalui media sosial.</li>
    </ul>
  </div>
</section>
