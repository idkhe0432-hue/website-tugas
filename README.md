<section id="jenis">
  <div class="card">
    <h2>Contoh Kejahatan Cyber</h2>
    <ul>
      <li><b>Phishing:</b> Penipuan dengan membuat website atau link palsu untuk mencuri username, password, atau data penting korban.</li>

      <li><b>Hacking:</b> Tindakan masuk ke akun atau sistem orang lain tanpa izin untuk mengambil atau merusak data.</li>

      <li><b>Malware:</b> Program berbahaya seperti virus atau trojan yang bisa merusak perangkat dan mencuri informasi.</li>

      <li><b>Penipuan Online:</b> Misalnya jual beli palsu di internet yang membuat korban kehilangan uang.</li>

      <li><b>Cyberbullying:</b> Perundungan melalui media sosial seperti menghina atau menyebarkan kebencian.</li>
    </ul>
  </div>
</section>
